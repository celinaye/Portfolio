Overview

The goal of the application is for users to view and learn facts about different animals. The information on the animals include the name of the animal, latin name, animal type (Amphibian, Bird, Mammal, or Reptile), the time the animal is active (Diurnal or Nocturnal), the maximum length of the animal in feet, and the maximum weight in pounds, the lifepan in years, the animal's habitat, diet, and geographical location.

API Selection

I selected a Zoo Animals API for my project. I received data about random zoo animals and their corresponding facts which I then converted into a json file so that I can access the data when building my application.

Architecture

The application allows the user to sort by the name of the animal with radio buttons from Ascending (A-Z), Descending (Z-A), or none at all. The user can also filter by the animal's active time and animal type with radio buttons simultaneously.

Additionally, user is able to add and remove an animal to their favorites and the total/aggregate maximum length of the animals in the favorites list updates accordingly. The user is also able to clear their favorites list with the "Clear Favorites" button.

Conclusion

From the project, I gained experience with building an application in React from scratch, while also taking into consideration usability principles. Specifically, I learned how React and React components work. That is, how data is passed down through components and how the user triggers state changes.
